J3DIO

J3DIO is a Java library for importing, storing, and exporting 3D model files. J3DIO also has limited support for rendering loaded models with OpenGL.

File type support

File Type Import Export Render
.obj Full Full Full
.mtl Partial Partial None
.raw Full Full Full
.ply Partial Partial None
.stl Partial Partial Partial
